Required Information
====================

* gender ("MALE"/"FEMALE")
* first name (str)
* name (str)
* mail (str)
* meter id (str)
* role ("LOCAL_POWER_TAKER"/"ADMINISTRATOR")
* inhabitants (int)
* flat size (float)
* group meter id (str)

Format
======

* e.g. as JSON object: 
{
  "gender" => str,
  "first_name" => str, 
  "name" => str, 
  "mail" => str, 
  "meter_id" => str,
  "role" => str, 
  "inhabitants" => int, 
  "flat_size" => float,
  "group_meter_id" => str
}
